# Fix Your Grammar

A simple Electron app that helps you fix grammar by selecting text and pressing `Alt+A` to open the app.

![App Demo](front.gif)

## Installation

```bash
npm install
npm run build
npm start
```

## Usage

- **Select text** → Press `Alt+A` → App opens with corrections
- **Double-click button** → Text gets copied and pasted automatically
- **Press `Alt+A` again** → App hides
